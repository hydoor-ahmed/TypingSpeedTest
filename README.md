---

## 🛠️ Tech Stack & Features
This solution was engineered using a modern front-end workflow:

* **Core:** HTML5, CSS3 (Tailwind CSS for styling), and Vanilla JavaScript / React.
* **State Management:** Real-time calculation of WPM (Words Per Minute) and Accuracy.
* **Data Handling:** Dynamic passage loading from local `data.json` based on difficulty levels.
* **Persistence:** `localStorage` integration to ensure your "Personal Best" survives browser restarts.
* **Animations:** Smooth visual feedback for correct/incorrect keystrokes and high-score celebrations.

---

## 🎮 Game Modes & Functionality

### 🕹️ Test Controls
- **Instant Start:** Jump straight into the action by typing or clicking the start button.
- **Difficulty Scaling:** Choose between **Easy**, **Medium**, and **Hard** passages.
- **Mode Switching:** - ⏱️ **Timed (60s):** Race against the clock.
    - 📖 **Passage Mode:** No time limit—focus on pure completion.
- **Quick Reset:** Instantly fetch a new passage and reset your stats.

### ⌨️ The Typing Experience
- **Real-time Feedback:** Visual cues (Green/Red) for every character typed.
- **Advanced Metrics:** Live tracking of WPM and accuracy percentage.
- **Error Recovery:** Support for backspacing to correct mistakes while maintaining integrity of the initial error count.

### 🏆 Results & Achievements
- **Baseline Establishment:** Your first run sets the bar.
- **High Score Smashed:** Custom celebration logic for when you outperform your previous self.
- **Persistence:** High scores are saved locally so you can always come back and beat them.
